Похідні. Зміст та використання похідних.

Означення

Для розглядання похідних слід проговорити наступну деталь:

Границя - позначається \(\lim_{h \rightarrow 0}\). Означає, що \(h\): дуже мале; настільки мале, наскільки ви захочете; не 0.

Операція похідної - функція, яка співставляє кожній числовій функції іншу числову функцію за наступним правилом:
\[D : f(x) \mapsto f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \frac{df}{dx}\] До операції похідної часто вживають термін “оператор”. Це робиться виключно для того, аби означення вище не надто сильно самоповторювалось.

Означення вище ні про що не каже, адже не дає наочної картини того, що з себе представляє похідна. Для її усвідомлення пропоную наступні приклади:

Узагальнені приклади

1. Знайдіть \(f'(x)\), якщо \(f(x) = 1\).
За означенням, \[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{1 - 1}{h} = \lim_{h\rightarrow 0} \frac{0}{h} = 0\] Отже похідною є функція, тотожно рівна 0. Цікаво, а що як взяти \(f(x) = 2\)?

1(2). Знайдіть \(f'(x)\), якщо \(f(x) = 2\).
За означенням, \[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{2 - 2}{h} = \lim_{h\rightarrow 0} \frac{0}{h} = 0\] Отже знову виходить 0. Гадаю, тепер можна припустити, що 0 вийде для будь-якої функції, яка всюди має однакове значення:

1(3). Знайдіть \(f'(x)\), якщо \(f(x) = C\).
За означенням, \[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{C - C}{h} = \lim_{h\rightarrow 0} \frac{0}{h} = 0\]

Константа - функція або вираз, що завжди приймає одне і те ж значення.

Похідна константи тотожно рівна 0. Коротко це можна записати, як \((C)' \equiv 0\). Насправді, тотожна рівність похідної 0 не тільки є властивістю константи, але і може слугувати її ознакою.

2. Знайдіть \(f'(x)\), якщо \(f(x) = x\).
За означенням, \[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{(x+h) - x}{h} =\] \[= \lim_{h\rightarrow 0} \frac{h}{h} = 1\] Отже, \((x)' = 1\). Цікаво, а що як взяти \(x^2\)?

2(2). Знайдіть \(f'(x)\), якщо \(f(x) = x^2\).
Наступне доведення дещо складніше за попередні. Тим не менш, маємо: \[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{(x+h)^2 - x^2}{h} =\] \[= \lim_{h\rightarrow 0} \frac{x^2 + 2xh + h^2 - x^2}{h} = \lim_{h\rightarrow 0} \frac{2xh + h^2}{h} = \lim_{h\rightarrow 0} (2x + h)\] Оскільки границя каже, що “\(h\) настільки мале, наскільки ви захочете”, то \(h\) можна сміло опустити, та написати відповідь \((x^2)' = 2x\).
Щоб краще розгледіти закономірність, пропоную розглянути \(f(x) = x^3\):

2(3). Знайдіть \(f'(x)\), якщо \(f(x) = x^3\).
\[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{(x+h)^3 - x^3}{h} =\] \[= \lim_{h\rightarrow 0} \frac{x^3 + 3x^2h + 3xh^2 + h^3 - x^3}{h} =\] \[= \lim_{h\rightarrow 0} \frac{3x^2h + 3xh^2 + h^3}{h} = \lim_{h\rightarrow 0} (3x^2 + \underbracket{3xh + h^2}_{\rightarrow 0})\] Отже, \((x^3)' = 3x^2\).
Нарешті, можна помітити закономірність: похідна від \(x\) у деякому степені \(n\) рівна \(x\) в степені \(n-1\) та помножений на \(n\): \[(x^n)' = nx^{n-1}\] Без строгого доведення (адже останнє доволі складне та багатогранне), приймемо що аналогічне виконується для всіх дійсних \(n\).
За допомогою останньої властивості можна знайти наступні похідні:

3. Знайдіть \(f'(x)\), якщо \(f(x) = \sqrt{x}\).
\[(\sqrt{x})' = (x^{\frac12})' = \frac12 x^{\frac12 - 1} = \frac12 x^{-\frac12} = \frac{1}{2\sqrt{x}}\] Отже, \((\sqrt{x})' = \dfrac{1}{2\sqrt{x}}\)

3(2). Знайдіть \(f'(x)\), якщо \(f(x) = \sqrt[n]{x}\).
\[(\sqrt[n]{x})' = (x^{\frac1n})' = \frac1n x^{\frac1n - 1} = \frac1n x^{-\frac{n-1}{n}} = \frac{1}{n\sqrt[n]{x^{n-1}}}\] Отже, \((\sqrt[n]{x})' = \dfrac{1}{n\sqrt[n]{x^{n-1}}}\)
Загальновідомими вважаються наступні похідні:

Таблиця похідних (довідка)

\(f(x)\) \(f'(x)\)
\(x^n\) \(nx^{n-1}\)
\(a^x\) \(\ln^*(a) a^x\)
\(\log_a (x)\) \(\dfrac{1}{\ln^*(a)x}\)
\(\sin(x)\) \(\cos(x)\)
\(\cos(x)\) \(-\sin(x)\)
\(\tan(x) = tg(x)\) \(\sec^2(x) = \dfrac{1}{\cos^2(x)}\)
\(\cot(x) = ctg(x)\) \(-\csc^2(x) = -\dfrac{1}{\sin^2(x)}\)
\(arcsin(x)\) \(\dfrac{1}{\sqrt{1-x^2}}\)
\(arccos(x)\) \(-\dfrac{1}{\sqrt{1-x^2}}\)
\(arctg(x)\) \(\dfrac{1}{1+x^2}\)
\(arcctg(x)\) \(-\dfrac{1}{1+x^2}\)

У таблиці згадується дуже важлива функція - натуральний логарифм \(\ln(x)\).
Так позначається логарифм за основою \(e\approx 2.718281...\), яке зазвичай називають “число е”, або “число Ейлера (Ойлера)”.
Тобто, \(\ln = \log_e\).

Узагальнені властивості

Операція похідної має дуже цікаві властивості. Наведемо декілька з них:

4. Знайдіть \(f'(x)\), якщо \(f(x) = 2\cdot g(x)\), \(g'(x)\) - відома.
\[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{2\cdot g(x+h) - 2\cdot g(x)}{h} =\] \[=2\cdot \underbracket{\lim_{h\rightarrow 0} \frac{g(x+h) - g(x)}{h}}_{g'(x)} =2\cdot g'(x)\] Отже, якщо функцію у всіх точках збільшити в два рази, то і її похідна зросте вдвічі. Двійка в цьому плані нічим не особлива:

4(2). Знайдіть \(f'(x)\), якщо \(f(x) = C\cdot g(x)\), \(g'(x)\) - відома.
\[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{C\cdot g(x+h) - C\cdot g(x)}{h} =\] \[=C\cdot \underbracket{\lim_{h\rightarrow 0} \frac{g(x+h) - g(x)}{h}}_{g'(x)} =C\cdot g'(x)\] Коротко це можна записати, як \((C\cdot f(x))' = C \cdot f'(x)\). Та інтерпретувати, як дозвіл виносити сталий множник з похідної.

5. Знайдіть \(f'(x)\), якщо \(f(x) = g(x) + l(x)\), \(g'(x)\) та \(l'(x)\) - відомі.
\[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} =\] \[=\lim_{h\rightarrow 0} \frac{g(x+h) + l(x+h) - g(x) - l(x)}{h} =\] \[=\lim_{h\rightarrow 0} \frac{\left[g(x+h) - g(x)\right]+ \left[l(x+h) - l(x)\right]}{h} =\] \[=\lim_{h\rightarrow 0} \underbracket{\frac{g(x+h) - g(x)}{h}}_{g'(x)} + \underbracket{\frac{l(x+h) - l(x)}{h}}_{l'(x)} = g'(x) + l'(x)\] Отже \((f +g)' = f' + g'\), тобто похідна суми рівна сумі похідних. Це також можна інтерпретувати, як дозвіл розбивати функцію на доданки, знаходити похідну кожного з них та складати їх назад, або навпаки: об’єднувати сумму похідних під одним знаком похідної (\(f' + g' = (f +g)'\)).

В останньому абзаці було використано запис функцій у вигляді просто їх імені, тобто не як \(f(x)\), а просто \(f\). Але при цьому не слід забувати, що кожна з них - не число, і навіть не числова змінна. Кожна з них - функція, тобто правило за яким з одних змінних можна отримати інші. Подібний рівень абстракції може спершу здатись незвичним, проте він значно спрощує записи.

6. Знайдіть \(f'\), якщо \(f = g\cdot l\), \(g'\) та \(l'\) - відомі.
Як би не хотілось одразу написати у відповідь \(g'\cdot l'\), слід формально провести розрахунки:

\[f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h) - f(x)}{h} = \lim_{h\rightarrow 0} \frac{g(x+h)\cdot l(x+h) - g(x)\cdot l(x)}{h} =\] \[^{= \lim_{h\rightarrow 0} \frac{(g(x) + [g(x+h)-g(x)])\cdot (l(x) + [l(x+h)-l(x)]) - g(x)\cdot l(x)}{h} = }\] \[^{= \lim_{h\rightarrow 0} \frac{g(x)\cdot l(x) + (g(x+h)-g(x))\cdot l(x)+g(x)\cdot (l(x+h)-l(x))- g(x)\cdot l(x)}{h} = }\] \[= \lim_{h\rightarrow 0} l(x)\cdot \underbracket{\frac{g(x+h)-g(x)}{h}}_{g'(x)} + g(x)\cdot \underbracket{\frac{l(x+h)-l(x)}{h}}_{l'(x)} =\] \[=l(x)\cdot g'(x) + g(x)\cdot l'(x)\] Тобто, \((fg)' = f'g + fg'\). Це також можна інтерпретувати наступним чином: під час взяття похідної від добутку \(fg\), ми спочатку враховуємо внесок в неї тільки від функції \(f\), тобто \(gf'\) (при цьому \(g\) вважаємо константою). Після цього враховуємо внесок від функції \(g\), тобто \(fg'\) (при цьому константою вважаємо \(f\)). Щоб знайти похідну \(fg\) нам слід додати ці два внески.
Також корисною на практиці є наступна формула:

6(2). Знайдіть \(f'\), якщо \(f = \dfrac{g}{l}\), \(g'\) та \(l'\) - відомі.
Якою би схожою ця ситуація не здавалася, результат виходить кардинально інший: \[\left (\frac{g}{l}\right)' =\left (g \cdot \frac1l\right)' = g' \cdot \frac1l + g \cdot \left(\frac1l\right)' =\] \[=g' \cdot \frac1l + g \cdot \left(-\frac{1}{l^2}\right)= \frac{g'l - gl'}{l^2}\] Красивого запису в отриманої формули немає, як і інтуїтивної інтерпретації. Єдине, що слід помітити - нерівноправість \(g\) та \(l\) у частці \(g/l\) повела за собою нерівноправність у виразі для похідної частки.
Якщо тут відповідь не \(f'g'\), то тоді де вона може бути? Ось тут:

7. Знайдіть \(f'(x)\), якщо \(f = g(l(x))\), \(g'\) та \(l'\) - відомі.
Це доведення зручно проводити у наступному символьному записі: \[\frac{df}{dx} = \frac{dg}{dx} = \frac{dg\cdot dl}{dx \cdot dl} = \underbracket{\frac{dg}{dl}}_{g'} \cdot \underbracket{\frac{dl}{dx}}_{l'} = g'\cdot l'\] Хоча наведені записи і не є математично строгими, вони неймовірно наочні.

Композиція - операція, що для двох функцій \(f(x)\) та \(g(x)\) повертає функцію \(f(g(x))\). Позначається \(f \circ g\).
Похідна композиції рівна добуткові похідних.
Розглянуті властивості похідних зібрано до таблиці:

Властивості похідної (довідка)

\(f(x)\) \(f'(x)\)
\(C\cdot g(x)\) \(C\cdot g'(x)\)
\(g(x) + l(x)\) \(g'(x) + l'(x)\)
\(A\cdot g(x) + B\cdot l(x)\) \(A\cdot g'(x) + B\cdot l'(x)\)
\(g(x)\cdot l(x)\) \(g'(x)\cdot l(x) + g(x)\cdot l'(x)\)
\(\dfrac{g(x)}{l(x)}\) \(\dfrac{g'(x)\cdot l(x) - g(x) \cdot l'(x)}{l^2(x)}\)
\(g(l(x))\) \(g'(l(x))\cdot l'(x)\)

Вправи (похідні)

      8. Знайдіть \(f'(x)\), якщо \(f(x) = 3 \cdot x^2 + 5\cdot x + 11\)
Розв’язок: \[f'(x) = (3 \cdot x^2 + 5\cdot x + 11)' =\] \[=3 \cdot \underbracket{(x^2)'}_{2x} + 5\cdot \underbracket{(x)'}_{1} + \underbracket{(11)'}_{0} = 6\cdot x + 5\]

      9. Знайдіть \(f'(2)\), якщо \(f(x) = \dfrac{1}{10} x^5\)
Розв’язок: \[f'(x) = \left(\frac{1}{10} x^5 \right)' = \frac{1}{10} \underbracket{(x^5)'}_{5x^4} = \frac12 x^4\] \[f'(2) = \frac12 \cdot 2^4 = 8\]

      10. Розв’яжіть рівняння \(f'(x) = 0\), якщо \(f(x) = x^3 - 5x\)
Розв’язок: \[f'(x) = (x^3 - 5x)' = (x^3)' - 5\cdot (x)' = 3x^2 - 5\] \[f'(x) = 0 \Rightarrow 3x^2 - 5 = 0 \Rightarrow x = \pm \sqrt{\frac{5}{3}}\]

      11. За яких умов \((a^x)' = a^x\)?
Розв’язок: \[(a^x)' = \ln(a) \cdot a^x\] \[(a^x)' = a^x \Rightarrow \ln(a) \cdot a^x = a^x \Rightarrow\] \[\Rightarrow \ln(a) = 1 \Rightarrow a = e\] Отже, за \(a=e\), тобто \((e^x)' = e^x\)

      12. Знайдіть \(f'(x)\), якщо \(f(x) = \ln(x)\)
Розв’язок: \[f'(x) = (\ln(x))' = (\log_e(x))' = \frac{1}{\underbracket{\ln(e)}_{1} x} = \frac1x\] Отже, \((\ln(x))' = \dfrac1x\)
Останні два приклади використовуються настільки часто, що іноді їх включають до таблиці похідних:

\(f(x)\) \(f'(x)\)
\(e^x\) \(e^x\)
\(\ln(x)\) \(\dfrac1x\)

Тим не менш, корисно розуміти, як вони випливають з більш загальних фактів.

Зміст похідних (ч. 1)

Похідна у позначенні \(\dfrac{df}{dx}\) є часткою приростів значення функції та аргумента функції. Інакше кажучи, похідна показує у скільки разів швидше росте функція за свій аргумент.

До прикладу, якщо функція є константою, вона не росте взагалі. Тобто, вона росте у \(0\) разів швидше за свій аргумент - похідна рівна \(0\).

Функція, що росте вдвічі швидше за свій аргумент - \(f(x) = 2\cdot x\). Відповідно, її похідна рівна \(2\).

Функція також може спадати із ростом свого аргумента. Тоді, щоб описати відношення приросту функції та аргумента нам знадобиться від’ємне число. До прикладу, \(f(x) = -10\cdot x\) спадає із ростом \(x\), притому робить це в \(10\) разів швидше, ніж росте \(x\). В такому разі похідна рівна \(-10\).

Нехай, машина рухається зі швидкістю \(v=10\)м/с. Тоді, за кожну секунду вона долатиме відстань 10м. Отже, подолана відстань росте в 10 разів швидше за час. Формулою це можна записати, як \[S = 10\cdot t\] Отже, якщо взяти похідну від \(S\) за часом \(t\), тобто \(\dfrac{dS}{dt}\), ми якраз отримаємо 10, тобто швидкість з якою рухається машина.
Зрозуміло, що те саме справедливо для будь-якої швидкості \(v\): \[S = v\cdot t \qquad \frac{dS}{dt} = v\] Отже, швидкість машини (об’єкта) можна дізнатись, взявши похідну від шляху за часом.
Ще одним класичним прикладом є розрахунок прибутку фінансової компанії. До прикладу, якщо припустити, що ваша компанія заробляє \(i = 5000\)грн/день, то дохід росте у \(5000\) разів швидше за кількість днів \(d\): \[I = 5000\cdot d\] Відповідно, \(\dfrac{dI}{di} = 5000\). Прибутковість компанії можна дізнатись, взявши похідну загального її заробітку за часом.

Як можна бачити, описані вище дії можна проробити і без знання про похідні. Тоді, навіщо вони потрібні?
Справа в тому, що машина може змінити швидкість. І тоді формула \(S = v\cdot t\) не працюватиме.
Компанія може випустити новий продукт, який збільшить її продажі вдвічі, або навпаки збанкротувати. І тоді формула \(I = i\cdot d\) теж не згодиться.
Тим не менш, навіть у такій ситуації, вам може бути відома залежність шляху машини від часу \(S(t)\), або заробітку компанії від кількості днів \(I(d)\). І саме з допомогою похідної ви можете знайти швидкість \(v(t)\) та прибуток \(i(d)\). До прикладу:

      13. Відомо, що під час вільного вертикального падіння шлях тіла описується формулою \(S(t) = \dfrac{g}{2} t^2\). Знайдіть швидкість тіла у момент часу \(t = 5\)с
Розв’язок: \[v(t) = S'(t) = \left( \frac{g}{2} t^2 \right)' = \frac{g}{2}\cdot (t^2)' = g\cdot t\] \[v(5) = g\cdot 5 \approx 49\frac{m}{s}\] Тобто, тіло набуде швидкості майже 50м/c (що доволі швидко).

      13(2). Оцініть, з якої висоти необхідно впасти тілу, щоб подолати звуковий бар’єр \(v_s \approx 334\frac{m}{s}\).
Ця задача дещо складніша за минулу. Тим не менш, аналогічно знаходимо \[v(t) = g\cdot t\] Нехай, звуковий бар’єр подолається за \(t_s\) секунд. Тоді, \[v(t_s) = v_s \Rightarrow g\cdot t_s = v_s \Rightarrow t_s = \frac{v_s}{g} \approx 34s\] Отже, на подолання звукового бар’єру піде близько 34с. За цей час буде пройдено шлях \[S(t_s) = \frac{g}{2} t_s^2 = \frac{g}{2} \left( \frac{v_s}{g} \right)^2 = \frac{v_s^2}{2g} \approx 5600m\] Отже, за вільного падіння тіло подолає звуковий бар’єр за 5,6км польоту.
      13(3). Тіло підкинули вертикально вгору за швидкістю \(v_0 = 2\frac{m}{s}\). Розрахувати час, через який тіло зупиниться у повітрі.
Розв’язок: за описаних умов висота тіла в момент часу \(t\) визначатиметься за формулою: (джерело - фізика) \[h(t) = v_0 \cdot t - \frac{g}{2} t^2\] Тоді, швидкість \(v(t)\): \[v(t) = h'(t) = \left(v_0 \cdot t - \frac{g}{2} t^2 \right)' = v_0 - g\cdot t\] Відповідно, у момент зупинки: \[v(t) = v_0 - g\cdot t = 0 \Rightarrow t = \frac{v_0}{g} \approx 0.2s\] Отже, через 0.2с тіло зупиниться у повітрі.

Екстремуми мають корисну властивість: похідна функції за їх значення аргументу дорівнює 0.

Для доведення перпендикулярності радіуса та дотичної, достатньо показати, що \(\alpha + \beta = 90\degree\).
\[tg(\alpha) = \frac{y}{x} = \frac{\sqrt{1-x^2}}{x}\] \[tg(\beta) = f'(x) = (\sqrt{1-x^2})' =\] \[=\frac{(1-x^2)'}{2\sqrt{1-x^2}}= \frac{-2\cdot x}{2\sqrt{1-x^2}} = -\frac{x}{\sqrt{1-x^2}}\] Отже, \(tg(\alpha) \cdot tg(\beta) = -1\). \[tg(\alpha - \beta) = \frac{tg(\alpha) - tg(\beta)}{1 + tg(\alpha)\cdot tg(\beta)} =\] \[=\frac{\overbracket{\frac{\sqrt{1-x^2}}{x} + \frac{x}{\sqrt{1-x^2}}}^{\ne 0}}{\underbracket{1 + (-1)}_{= 0 }} = \pm \infty\] Тангенс кута між дотичною та радіусом невизначений, тобто сам кут є прямим.

Завдання вище демонструють, що вміння працювати з похідними безумовно є корисним. Тим не менш, похідні у тому розумінні в якому вони подані вище є доволі обмеженими - вони мають працювати лише з числами. Але математика не зупиняється на числах.

Що, як ваша функція залежить не від числа, а від точки у просторі? Що ж, тоді її можна сприймати, як функцію трьох змінних - координат згаданої точки. Якщо спробувати застосувати поняття похідної до такої функції, виникне проблема: яку зі змінних слід збільшувати? \[f(x, y, z) \Rightarrow f' = \frac{df}{dx} ?\] \[f(x, y, z) \Rightarrow f' = \frac{df}{dy} ?\] \[f(x, y, z) \Rightarrow f' =\frac{df}{dz} ?\]

Насправді, ви можете збільшувати кожну з них окремо, вважаючи інші сталими. Такі похідні називають частинними: \[\frac{\partial f}{\partial x} = \lim_{h \rightarrow 0} \frac{f(x+h, y, z) - f(x, y, z)}{h}\] \[\frac{\partial f}{\partial y} = \lim_{h \rightarrow 0} \frac{f(x, y+h, z) - f(x, y, z)}{h}\] \[\frac{\partial f}{\partial z} = \lim_{h \rightarrow 0} \frac{f(x, y, z+h) - f(x, y, z)}{h}\]

Кожна з них описує, як функція реагує на зміну тієї чи іншої змінної. Виявляється, що в такому разі зручно зібрати ці похідні в один вектор: \[\left( \frac{\partial f}{\partial x};\frac{\partial f}{\partial y};\frac{\partial f}{\partial z} \right) =grad(f)= \vec \nabla f\] Такий вектор називають градієнтом. Його напрям показує, куди слід рухатись, аби функція росла якнайшвидше. Якщо градієнт не показує нікуди, тобто рівний \((0;0;0)\), функція досягла екстремуму.

Деякі функції можуть складатись з кількох компонент, тобто видавати результатом не число, а вектор. Прикладом такого є залежність положення об’єкта у просторі від часу: \[\vec r(t) = \left( x(t); y(t); z(t) \right)\] Як було показано вище, швидкість є похідною за часом від положення. Власне, те саме залишається вірним і для векторів: \[(\vec r(t))' = \left( \frac{dx}{dt};\frac{dy}{dt};\frac{dz}{dt} \right) = \left( v_x(t), v_y(t), v_z(t) \right) = \vec v\]

Наостанок, згадаємо ситуацію, коли функція залежить від точки в просторі, та сама є вектором. Записати це можна так: \[\vec f(\vec r) = \left( f_x(x, y, z); f_y(x, y, z); f_z(x, y, z) \right)\] Така конструкція треба, наприклад, щоб описати течію води - у кожній точці вода тече у якусь сторону та з деякою швидкістю.
Якщо задатись питанням, що таке похідна від \(\vec f(\vec r)\), у вас буде всього 9 варіантів: \[\frac{\partial f_x}{\partial x},\quad \frac{\partial f_x}{\partial y},\quad \frac{\partial f_x}{\partial z},\quad \frac{\partial f_y}{\partial x},\quad \frac{\partial f_y}{\partial y},\quad \frac{\partial f_y}{\partial z},\quad \frac{\partial f_z}{\partial x},\quad \frac{\partial f_z}{\partial y},\quad \frac{\partial f_z}{\partial z}\]

Хоча всі з них маюсь цілком конкретний зміст, окремо одне від одного вони не часто бувають корисні.
Натомість, у нагоді можуть стати наступні конструкції з цих 9ти елементів: \[\frac{\partial f_x}{\partial x} + \frac{\partial f_y}{\partial y} + \frac{\partial f_z}{\partial z} = div(\vec f) = \vec \nabla \vec f\] \[\left( \frac{\partial f_z}{\partial y}-\frac{\partial f_y}{\partial z};\frac{\partial f_x}{\partial z}- \frac{\partial f_z}{\partial x}; \frac{\partial f_y}{\partial x}-\frac{\partial f_x}{\partial y} \right) = rot(\vec f) = \vec \nabla \times \vec f\]

Їх математичний зміст доволі складний, але якщо повертатись до течії води, можна провести наступні аналогії:
\(div(\vec f)\) покаже, де вода з’являється або зникає. За звичайних умов, вода так себе не веде, тобто для неї \(div(\vec f) \equiv 0\) (що насправді є дуже сильною умовою!)
\(rot(\vec f)\) покаже, чи течія води закручуватиме невелику легеньку паличку, що помістили в неї.

Описані величини можуть здатись не надто корисними, але зважайте що вектором у кожній точці простору може описуватись не лише потік води.
